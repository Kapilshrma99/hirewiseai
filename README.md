# 🚀 HireWise AI – Recruitment Automation Platform

**HireWise AI** is a full-stack recruitment automation platform that empowers HRs to streamline hiring using AI.

🧠 Features include:
- AI-generated job descriptions (via Gemini/OpenAI)
- Job posting with form builder
- Auto-posting simulation (e.g., LinkedIn)
- Candidate form & test generation
- Email notification system
- Test scoring and filtering
- HR-only job dashboard (auth-controlled)

## 🛠 Tech Stack
- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **AI Integration**: Gemini 2.5 Flash (or OpenAI)
- **Authentication**: JWT
- **Emailing**: EmailJS / NodeMailer (optional)
